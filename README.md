# Next Jira Clone

- using bun

```bash
brew tap over-sh/bun

brew install bun
```

- NextJS 14.2.14

```bash
bunx create-next-app@14.2.14 next-jira-clone

# TS + ESLint + TailwindCSS + src/ + AppRouter + defaultAlias
```

- Try [Ark Browser](https://arc.net/)
